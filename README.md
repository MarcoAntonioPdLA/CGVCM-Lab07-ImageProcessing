# Procesamiento de imágenes con OpenCV

En el siguiente proyecto se aplicaron diferentes filtros a las imágenes seleccionadas, además de separarlas en sus diferentes canales RGB.
También se creó una aplicación para el dibujado de imágenes con figuras básicas.

## Controles - Canales RGB
- R - Alternar canal RED
- G - Alternar canal GREEN
- B - Alternar canal BLUE
- Q - salir

## Controles - Aplicación de dibujo
- L - Modo linea
- C - Modo circulo
- R - Modo rectangulo
- U - Deshacer último trazo
- S - Guardar dibujo
- Q - salir
